# SyZero Service CLI
[![Latest npm release][npm-badge]][npm-badge-url]
## 介绍
syzero服务脚手架

## 安装

```bash
npm install -g syzero-service-cli
```

## 用法

安装后，CLI 工具将可供您使用。它是上述所有功能的入口点。
```bash
syzero
```
根据提示创建你的第一个SyZero项目！
